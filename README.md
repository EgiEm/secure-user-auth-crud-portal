# 🔒 Secure User Authentication & CRUD Portal

A full-stack user authentication and database management portal built with PHP and MySQL. It demonstrates security practices such as password hashing, persistent session handling, and access control combined with an administrative dashboard.

## 🚀 Features

- **Secure Login & Registration**: Input sanitization and encrypted password verification.
- **Session-Guarded Access**: Admin portal access restricted to validated logged-in users.
- **User CRUD Control**: Comprehensive dashboard panel to add, edit, view, and delete user profiles.
- **Database Schema**: Simple configuration utilizing the provided `db.sql` layout.

## 🛠️ Technologies Used

- **Backend**: PHP, MySQL
- **Frontend**: HTML5, CSS3, JavaScript (ES6)
- **Database**: SQL
